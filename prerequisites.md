# Prerequisites

- M270 Platform
- BIOS Password (WatchGaurd!)
- USB to Ethernet Adapter
- Cisco Style Console Cable (RJ45) to USB Serial Adapter.  Optional: You can use a Cisco Style Console Cable (RJ45) to Serial Port Adapter if you prefer
- Serial Software such as PuTTY set to 115200,N,8,1
- FreeBSD RELEASE or STABLE 15.0 or 15.1 memstick .img and flashed to USB (Please do NOT flash the ISO using Rufus or try to load the ISO using VenToy as this will not work.  You can use dd from a NIX machine.)
